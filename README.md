# README

## Requirement

- Node.js
- Python 2.7

## How to edit

Install some libraries

```bash
# required python libraries for mkdocs
$ pip install mkdocs
$ pip install fontawesome_markdown
$ pip install pygments

# lint
$ npm install
```

Checkout `source` branch

```bash
$ git checkout source
```

Launch local server, and access to http://127.0.0.1:8000

```bash
$ mkdocs serve
```
